# 更新记录

## v50 / 1.2.0

- 重做数据页为“趋势｜月历｜汇总”：默认 7 天，可切换 30/90 天，每次仅显示一个主分类和主图。
- 新增体重、体脂、围度快捷记录入口；训练统计加入周总组数、周总容量、真实动作趋势、分部位最佳成绩和 Epley 估算 1RM。
- 新增月度真实汇总与训练月历：日格显示碳日、训练部位组合、休息或自定义事项，休息与无记录明确分离，底部仅展示选中日期详情。
- 饮食页整合“今日饮食｜食物库｜补剂库”互斥视图；“我”页移除补剂管理，恢复页仅保留今日补剂记录。
- 成就系统扩展为 48 项铜/银/金/钻石阶梯成就和 8 项隐藏成就，全部使用真实训练、饮食、饮水、睡眠及身体数据计算进度。
- 固定碳日联动规则：背或腿为高碳，胸或肩为中碳，手臂、腹部、有氧或明确休息为低碳；复合训练按高、中、低优先级判断，无记录不会被当作休息。
- 新增历史训练组合复用：可按整场训练组合复用计划，并支持动作库按常练、最近、名称排序。
- 增强组间休息控制：支持 -10 秒、+10 秒、暂停/继续和跳过；自然结束只提醒一次，暂停、调整和跳过会取消或重排旧提醒。
- Android 休息提醒采用原生 BroadcastReceiver + AlarmManager，可在锁屏、后台及 APP 进程被系统回收后直接发送通知，并遵守系统静音、勿扰模式和通知渠道设置。用户主动强行停止 APP 后，Android 会禁用其闹钟，需重新打开 APP 才能恢复；手机重启后的提醒恢复不在本版本范围。该能力仍需在 iQOO 11S 真机上完成最终验收。
- 优化训练中操作契约：完成组可撤销后重新调整重量和次数，关键训练按钮与图标按钮保持 48dp 触控尺寸。
- 训练结束增加统一防误触确认，包括全部组已完成的情况；暂停休息减到 0 秒会立即结束并提醒。
- 完善 sessions 数据摘要：同一天多场训练可独立归档、去重汇总，并保留当前训练摘要口径。
- 明确身体数据口径：体重、体脂可分别标记实测，围度单独记录；沿用值不再被当作趋势样本。
- 完成 P0 可读性改造：正文、关键数值和次要说明建立三级层次，提高字号、字重与间距；正文按 WCAG AA 思路提升对比度，阅读卡、输入框、图表及训练按钮使用不透明实色，轻毛玻璃仅保留在导航和弹窗层级。

## v49 / 1.1.0

- 新增 106 个内置训练动作，支持部位筛选、名称/器械/目标肌群搜索、动作要点和常见错误说明
- 新增动作级训练流程：逐组重量与次数、完成状态、组间计时、训练总结及上次训练复用
- 支持同一天保存多场训练，修复新训练覆盖已完成训练的问题
- 增加未完成训练结束确认、连续点击防误记、旧训练兼容和跨日训练恢复
- 全局输入框改用独立上方标签，并统一并排输入的标签高度、输入框高度与顶部对齐
- 精简身体与恢复页，仅保留当天体重体脂、饮水、补剂和睡眠记录
- 增加原子化 JSON 写入和可指定测试数据目录，降低异常退出造成的数据损坏风险
