# 碳水大王 v44（基于 v43）

## 本次调整
- 新增备份导入：支持完整备份、历史记录、食物库、补剂库和个人资料 JSON
- 导入时可选择“合并导入”或“覆盖导入”，操作前自动保存本地安全快照
- “我”页新增宏量目标“自动计算 / 自定义”模式，可编辑高中低碳的碳水、蛋白质、脂肪倍数
- 历史详情支持上下滑动
- 小睡改为点击“添加小睡”后在弹窗中填写
- 今日默认沿用上一次更新的体重、体脂，“我”页同步显示
- 新增完整备份、历史记录、食物库、补剂库、个人资料导出
- 固定 Android 包名并自动递增构建号，保留同一签名后可覆盖更新且不清除记录
- 保留 v42 的目标区间紧凑整数显示修复

## 打包
双击 `build_apk_fast.bat`。脚本固定使用包名 `com.chenyang.carbs_king`，每次成功打包后自动准备下一个构建号。

第一次成功打包后，签名密钥会备份到：

`%USERPROFILE%\.carbs_king_signing\debug.keystore`

不要删除这个备份；后续脚本会固定使用它签名。包名或签名不同，Android 会当成另一个 APP；构建号必须逐次增加。

如果手机上现有版本使用了不同包名或旧签名，只能最后重装一次；从固定包名和签名的版本以后即可直接覆盖更新并保留数据。
